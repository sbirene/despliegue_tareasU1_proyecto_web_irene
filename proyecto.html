Tarea II: Hacer llegar el proyecto

1. Conectase con ftp o ssh desde el cliente de desarrollo al servidor de despliegue.

2. Dejar la web en una carpeta en la que pueda verse más tarde.

3. Otorgar al proyecto los permisos necesarios.

4. Hacer una captura de pantalla de la web funcionando.
